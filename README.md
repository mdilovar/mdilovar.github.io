# `rubik_n`

An online 3D Rubik's Cube game.
Players can choose cubes of various sizes (2x2x2 - 10x10x10).

Play now - https://rubikn.herokuapp.com

Uses three.js.

Video demo - https://youtube.com/watch?v=eFspplMmheU
